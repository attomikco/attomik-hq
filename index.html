Setup Guide

Follow these steps to connect your Google Sheet:

  1. Go to Google Sheets and create a new spreadsheet. Name it Invoice Manager.
  2. In the menu, click Extensions → Apps Script.
  3. Delete the default code and paste the entire contents of Code.gs.
  4. Click Save, then click Deploy → New deployment.
  5. Choose type: Web app. Set "Execute as" = Me, "Who has access" = Anyone. Click Deploy.
  6. Copy the Web app URL and paste it into the connection box on this app.
  7. On first run, Google will ask for authorization — click Allow.

✅ This app is hosted on GitHub Pages at attomikco.github.io/attomik-invoice-manager — CORS is fully supported from this origin.
