Setup Guide
  1. Go to Google Sheets and create a spreadsheet named Invoice Manager.
  2. Click Extensions → Apps Script, paste the Code.gs content, and save.
  3. Click Deploy → New deployment → Web app → Execute as: Me → Anyone → Deploy.
  4. Copy the Web App URL and paste it in the connection box.
